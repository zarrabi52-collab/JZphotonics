1. Unzip the package.
2. Open index.html in a browser.

How to publish for free:
- Upload the whole folder to GitHub Pages, Cloudflare Pages, or Netlify.
- Make sure the assets/images folder remains in the same structure.

To add your real photo:

2. Name it joseph-zarrabi.jpg
3. In index.html, replace:
   assets/images/joseph-zarrabi-placeholder.svg
   with:
   assets/images/joseph-zarrabi.jpg

Form hosting note:
- The form is configured for Netlify Forms.
- If using Netlify, upload this folder and the form should work automatically.
- If using GitHub Pages or Cloudflare Pages, use Formspree or another form service and replace the form action URL.
